Project Ideas

15 projects sorted by difficulty — the fastest way to truly learn is to build

How to use this list: After finishing each section of your guides, pick a project from that difficulty tier. Don't skip to advanced too fast — building beginner projects perfectly teaches more than rushing advanced ones.
Beginner

Tier 1 — Beginner Projects

Pure HTML + CSS. No JavaScript needed. Focus on layout, spacing, and responsive design.

01

Personal Portfolio Page

A single-page portfolio with your name, a short bio, skills section, and contact info. The classic first project — every developer has one.

HTML Semantics Flexbox CSS Variables Media Queries
Challenge: Make it fully responsive — looks great on mobile AND desktop.
02

Product Landing Page

A landing page for a fake product — hero section, features grid, pricing table, and a footer. Like an Apple product page but simpler.

CSS Grid grid-template-areas Transitions Typography
Challenge: Use CSS Grid for the features section with at least 3 columns.
03

Blog Post Layout

A styled blog article page with a sidebar. Real content, proper heading hierarchy, a table of contents in the sidebar, and related posts.

HTML Semantics CSS Grid (sidebar) Typography Accessibility
Challenge: Use proper semantic tags — article, aside, nav, header, footer.
04

Pricing Card UI

Three pricing tier cards (Free, Pro, Enterprise) with a highlighted "Popular" card. Hover effects and a feature checklist on each card.

Flexbox CSS Variables Pseudo-classes Box Shadow
Challenge: The popular card should be slightly bigger and have a glow effect using CSS only.
05

Animated Loading Screen

A stylish loading animation — spinning logo, progress bar, or pulsing dots. Pure CSS animations only, no JavaScript.

@keyframes CSS Animations Transforms Timing functions
Challenge: Build 3 different animation styles and combine them on one page.
Intermediate

Tier 2 — Intermediate Projects

HTML + CSS + JavaScript. DOM manipulation, events, arrays, and local storage.

06

Todo App (Full)

The most important beginner JS project. Add, complete, delete, and filter todos. Saves to localStorage so data persists after refresh.

DOM Manipulation Events Arrays Local Storage
Challenge: Add filter buttons — All / Active / Completed. Show count of remaining todos.
07

Weather App

Search a city, fetch real weather data from OpenWeatherMap API, display temperature, humidity, and conditions. Handle loading and error states.

Fetch API Async/Await Error Handling DOM + Events
Challenge: Show a 5-day forecast. Change the background based on weather conditions (sunny, rainy, cloudy).
08

Quiz App

A multiple-choice quiz with a timer, score tracking, and a results screen. Questions stored as an array of objects. Shows correct/wrong feedback.

Arrays & Objects DOM Manipulation setInterval (timer) Control Flow
Challenge: Randomize question order. Show a summary of wrong answers at the end with the correct answers.
09

Recipe Finder

Search recipes by ingredient using TheMealDB API. Display results as cards. Click a card to see full recipe details on the same page.

Fetch API Array Methods DOM Rendering Event Delegation
Challenge: Add a "Save Favourites" feature using localStorage. Show a favourites panel.
10

Expense Tracker

Add income and expense entries. See running total, balance, and a history list. Color-coded (green for income, red for expense). Saves to localStorage.

Array Methods Forms & Events Local Storage reduce()
Challenge: Add a category filter. Show a simple bar chart using only CSS (no library).
Advanced

Tier 3 — Advanced Projects

Full JS mastery — modules, APIs, complex state, clean architecture.

11

GitHub Profile Finder

Search any GitHub username via the GitHub API. Show avatar, bio, repos, followers, following. List top repositories with stars and language badges.

GitHub REST API Async/Await Modules Error Handling
Challenge: Split into modules — api.js, ui.js, utils.js. Add debounce to the search input so it doesn't fire on every keystroke.
12

Kanban Board

A drag-and-drop task board with columns (To Do, In Progress, Done). Add, edit, delete cards. Save state to localStorage. Full drag events.

Drag & Drop API Closures Local Storage Complex DOM
Challenge: Add card priority levels (high/medium/low) with color indicators. Allow renaming column titles.
13

Movie Search App

Search movies via the OMDB API. Show results in a grid. Click for a details modal — plot, cast, rating. Add to a watchlist saved in localStorage.

Fetch + API Keys Modules Modal UI Debouncing
Challenge: Add pagination — "Load More" button or infinite scroll. Rate movies from the watchlist.
14

Markdown Note App

A note-taking app that renders Markdown in real time. Sidebar of saved notes, click to edit, auto-save to localStorage. Use a Markdown library via CDN.

Modules Local Storage Real-time DOM Third-party library
Challenge: Add note titles and search/filter across note content. Add a dark/light mode toggle saved to localStorage.
15

Full Dashboard UI

A data dashboard with sidebar navigation, multiple panels (charts, tables, cards), dark/light mode, and collapsible sidebar. The ultimate CSS + JS project.

CSS Grid Layout CSS Variables Modules Chart.js (CDN)
Challenge: Fetch real data from a public API and populate the dashboard. Add a date range filter that updates the charts.

💡 How to Actually Learn From Projects

Start Without Looking

Try to build it from memory first. Only look at the guide when you're genuinely stuck — not to copy code.

Break It Into Pieces

Don't try to build everything at once. Write one function. Make it work. Then the next. Small wins compound.

Make It Ugly First

Get the functionality working first, then style it. Chasing perfect CSS before logic works is the #1 beginner trap.

Add Your Own Feature

Once the base project works, add one feature you came up with yourself. That's when real learning happens.

Put It On GitHub

Push every project to a GitHub repo. After 15 projects you have a real portfolio. Employers look at this.

Rebuild It 30 Days Later

Come back to a finished project a month later and rebuild it from scratch without looking. You'll be amazed how much you remember.
